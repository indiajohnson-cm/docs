Mga Opsyon at Tanda¶

Mga Argumento¶

-h, --help

Ipakita ang help message at mag-exit.

--images <path>, -i <path>

Kawing patungo sa mga imahe.

--project-path <path>

Kawing patungo sa mga proyekto.

--resize-to <integer>

Binabago ang laki ng imahe mula sa pinakamalaki na sukat para sa opensfm. I-set sa -1 para ma-disable.

Default: 2048

--end-with <string>, -e <string>

Opsyon: dataset | split | merge | opensfm | mve | odm_filterpoints | odm_meshing | mvs_texturing | odm_georeferencing | odm_dem | odm_orthophoto

--pc-ept

Generates Entwine Point Cloud. Ang flag na ito ay automatikong nadadagdag kapag sinimulan ang WebODM at magagamit kapag bumubuo ng tamang directory structure kapag nagpapatakbo ng NodeODM o sa pagimport ng command line para maging WebODM.

--rerun <string>, -r <string>

Opsyon: dataset | split | merge | opensfm | mve | odm_filterpoints | odm_meshing | mvs_texturing | odm_georeferencing | odm_dem | odm_orthophoto

--rerun-all

Pagulit sa lahat ng tasks.

--rerun-from <string>

Opsyon: dataset | split | merge | opensfm | mve | odm_filterpoints | odm_meshing | mvs_texturing | odm_georeferencing | odm_dem | odm_orthophoto

--proj <PROJ4 string>

Prodyeksyon na ginagamit sa pagtransform ng model sa mga heyograpikong coordinate.

--min-num-features <integer>

Pinakamababang bilang ng pwede i-extract na imahe. Mas maraming tampok ay magreresulta ng mas magandang resulta ngutin mas mabagal na pagsasakatuparan.

Default: 8000

--feature-quality

I-set ang feature extraction quality. Ang mas mataas na quality ay naggegenerate ng mas magandang features, ngunit ang mas kumakain ito ng mas malaking memorya at mas matagal ang proseso.

Mga Opsiyon: ultra | high | medium | low | lowest

Default: high

--matcher-neighbors <integer>

Bilang ng pinakamalapit na imahe na i-prematch base sa GPS exif data. I-set sa 0 para malagpasan anf pre-matching. Ang Neighbors ay gumagana sa pamamagitan ng Distance parameter, i-set pareho sa 0 para hindi na gamitin ang pre-matching. Ang OpenSFM ay gumagamit ng parehong parameters ng sabay, Ang Bundler naman ay gumagamit lamang ng isa na may value, mas preferred nito ang Neighbors parameter.

Default: 8

--matcher-distance <integer>

Ang distance threshold sa meters para hanapin ang pre-matching na imahe base sa GPS exif data. I-set pareho sa matcher-neighbors at 0 ito para malagpasan ang pre-matching.

Default: 0

--use-fixed-camera-params

Patayin ang camera paremeter optimization habang ginagamit ang blunder. Off ang default maliban kung --camera parameter ang gamit.

--camera-lens <string>

i-set ang klase ng camera projection. Ang manual na pagaayos ng value ay makakatulong sa pagpapa-improve ng geometric undistortion. By default, ang aplikasyon ay sinusubukan na i-determine ang lens type galing sa image metadata.

Mga Opsiyon: auto | perspective | brown | fisheye | spherical

Default: auto

--radiometric-calibration <string>

Magset ng radiometric calibration na gagamitin sa mga imahe. kapag nagpaprocess ng multispectral images, kailangan na i-set ang opsiyon para maobtain ang reflectance values (kung hindi ay makukuha lamang ang digital number (DN) values).

[camera] ay nagaapply ng black level, vignetting, row gradient gain/exposure compensation (kung sapat na EXIF tags ang mayroon).

Ang [camera+sun] ay experiemental, inaapply lahat ng correction ng [camera] at dinadagdagan ng spectral radiance na registered via a downdwelling light sensor (DLS) na pinagsasaalang-alang ang anggulo ng araw.

Mga Opsiyon: none | camera | camera+sun

Default: none

--max-concurrency <positive integer>

Ang pinakamalaking bilang ng proseso na gagamitin sa ibat-ibang proseso. Ang peak memory requirement ay ~1GB per thread at 2 megapixel na resolusyon ng imahe.

Default: number of cores

--depthmap-resolution <positive float>

Maaaring macontrol ang density ng point cloud sa pamamagitan ng pag-set ng resolusyon ng depthmap images. Ang mas mataas na value ay mas matagal macompute at mas malaking memorya ang mapo-produce ng denser point clouds.

Default: 640

--opensfm-depthmap-min-consistent-views <integer: 2 ≤ x ≤ 9>

Pinakakonti na bilang ng view na dapat magreconstruct sa point para ito ay maging valid. Gumamit ng mas mababang values kung ang imahe ay may mas konti na overlap. Ang mas mababang values ay nagreresulta ng mas dense na point clouds ngunit mas maraming noise. Ito ay nagaapply lamang kung ikaw ay gumagamit ng OpenSfM para sa dense matching.

Default: 3

--opensfm-depthmap-method <string>

Raw depthmap computation algorithm. PATCH_MATCH at PATCH_MATCH_SAMPLE ay mas mabilis, ngunit baka maka-miss ng valid points. Ang BRUTE_FORCE ay mas matagal ngunit nagreresulta ng desnser reconstructions.

Mga Opsiyon: PATCH_MATCH | PATCH_MATCH_SAMPLE | BRUTE_FORCE

Default: PATCH_MATCH

--opensfm-depthmap-min-patch-sd <positive float>

Kapag gumagamit ng PATCH_MATCH o PATCH_MATCH_SAMPLE, ang nagcocontrol ng standard deviation threshold para maisama ang mga patches. Ang mga patches na may mas mababang standard deviation ay ini-ignore.

Default: 1

--use-hybrid-bundle-adjustment

Mag-run ng local bundle adjustment kada imahe na dinadagdag sa reconstruction at global adjustment kada 100 na imahe. Pinapabilis nito ang reconstruction ng mga malalaking datasets.

--mve-confidence <float: 0 ≤ x ≤ 1>

I-discard ang mga points na may hindi siguradong threshold. Makakaapekto lamang ito sa dense reconstruction ng MVE. Ang higher values ay pinapagsawalang bahala ang mga points.

Default: 0.6

--use-3dmesh

Gumamit ng full 3d mesh upang macompute ang orthophoto kesa sa 2.5D mesh. Ang opsyon na ito ay mas mabilis at nagbibigay ng resulta sa planar areas.

--skip-3dmodel

Lagpasan ang generation ng full 3d model. Makakatipid ka ng oras kung ang resulta na kailangan mo lang ay 2D tulad ng orthophotos at DEMs.

--use-opensfm-dense

Buksan ang opensfm para macompute ang dense point cloud alternatively.

--ignore-gsd

Wag pansinin ang Ground Sampling Distance (GSD). Ang GSD ay nagcacap ng pinakamaraming bilang ng imahe at nagreresize ng imahe kung kinakailangan na nagreresulta ng mas mabilis na pagproseso at mas kaunti na memory usage. Bilang ang GSD ay estimate lamang, ang hindi pagbigay pansin dito ay magreresulta ng medyo mas mabuting quality ng imahe.

--mesh-size <positive integer>

Ang pinakamaraming bilang ng vertex ng output mesh.

Default: 100000

--mesh-octree-depth <positive integer>

Oct-tree depth used in the mesh reconstruction, increase to get more vertices, recommended values are 8-12.

Default: 9

--mesh-samples <float ≥ 1.0>

Bilang ng mga points kada octree node. Para sa mga low noise, ang value na nasa [1-5] ay magagamit. Para naman sa mas mataas na noise datasets, ang malaking value na nasa [15-20] naman ang magagamit.

Default: 1.0

--mesh-point-weight <positive float>

Tinutukoy ng floating point value ang kahalagahan na ang pag-interpolate ng mga muwestra ng mga punto ay ibinigay sa pagbabalangkas ng na-screen na Poisson equation. Ang mga resulta ng orihinal (unscreened) Poisson Reconstruction ay maaaring makuha sa pamamagitan ng pagtatakda ng halagang ito sa "0".

Default: 4

--fast-orthophoto

** Eksperimental. ** Nilaktawan ang masinsin na pagyari at pagbuo ng modelong 3D. Bumubuo ito ng isang orthophoto nang direkta mula sa kalat-kalat na pagyari. Kung orthophoto lamang ang kailangan, at hindi isang buong 3D na modelo, i-on ang opsyong ito.

--crop <positive float>

Awtomatikong i-crop ang mga imahe sa pamamagitan ng paglikha ng isang makinis na buffer sa paligid ng mga hangganan ng dataset, na lumiliit ng "N" na metro. Gamitin ang "0" para hindi paganahin ang pag-crop.

Default: 3

--pc-classify

Uriin ang mga point cloud output gamit ang isang Simple Morphological Filter. Maaari mong kontrolin ang mga katangian ng opsyong ito sa pamamagitan ng pag-aayos ng mga parametrong --dem-*.

Default: False

--pc-csv

I-export ang georeferenced point cloud sa CSV format.

Default: False

--pc-las

I-export ang georeferenced point loud sa LAS format.

Default: False

--pc-filter <positive float>

Sinasala ang cloud point sa pamamagitan ng pag-alis ng mga puntos na lumihis nang higit sa N standard deviation mula sa local mean. Itakda sa "0" upang hindi paganahin ang pag-filter.

Default: 2.5

--smrf-scalar <positive float>

Simple Morphological Filter elevation scalar parameter.

Default: 1.25

--smrf-slope <positive float>

Simple Morphological Filter slope parameter (rise over run).

Default: 0.15

--smrf-threshold <positive float>

Simple Morphological Filter elevation threshold parameter (meters).

Default: 0.5

--smrf-window <positive float>

Simple Morphological Filter window radius parameter (meters).

Default: 18.0

--texturing-data-term <string>

Mga Opsiyon: area | gmi

Default: gmi

--texturing-nadir-weight <integer: 0 ≤ x ≤ 32>

Mga orthophoto lang ang naaapektohan. Ang mga mas mataas na halaga ay nagreresulta sa mas matalas na sulok, ngunit maaaring makaapekto sa pamamahagi ng kulay at pagkalabo. Gumamit ng mas mababang halaga para sa mga planar area at mas mataas na halaga para sa mga urban area. Ang default na halaga ay mabuti para sa karamihan ng mga sitwasyon.

Default: 16

--texturing-outlier-removal-type <string>

Klase ng photometric outlier removal method.

Mga Opsiyon: none | gauss_damping | gauss_clamping

Default: gauss_clamping

--texturing-skip-visibility-test

Lagpasan ang geometric visibility test.

Default: False

--texturing-skip-global-seam-leveling

Lagpasan ang global seam leveling. Magagamit ito sa IR data.

Default: False

--texturing-skip-local-seam-leveling

Lagpasan ang local seam blending.

Default: False

--texturing-skip-hole-filling

Lagpasan ang pagpuno ng butas sa mga mesh.

Default: False

--texturing-keep-unseen-faces

Itago ang mga mukha sa mesh na hindi makikita sa camera.

Default: False

--texturing-tone-mapping <string>

I-turn on ang gamma tone mapping o wag para sa no tone mapping.

Mga Opsiyon: gamma | none

Default: none

--gcp <path string>

Ang path na naglalaman ng ground control points gamit sa georeferencing. Ang file ay kailangan ng mga sumusunod na line format: easting northing height pixelrow pixelcol imagename

Default: None.

--use-exif

Gamitin ang tag na ito kung mayroong gcp_list.txt ngunit gustong gumamit ng exit geotags.

--dtm

Gamitin ang tag na ito para bumuo ng DTM (Digital Terrain Model, ground only) gamit ang simpleng morphological filter. I-check ang --dem* at --smrf* na parameters para sa mas-finer na tuning.

--dsm

Gamitin ang tag na ito para bumuo ng DSM (Digital Surface Model, ground + objects) gamit ang PROGRESIBONG morphological filter. I-check ang --dem* na parameters para sa mas-finer na tuning.

--dem-gapfill-steps <positive integer>

Bilang ng mga hakbang na ginamit upang punan ang mga lugar ng mga puwang. Itakda sa "0" upang hindi paganahin ang pagpuno ng puwang. Magsimula sa isang radius na katumbas ng resolusyon ng output, N ang bilang ng iba't ibang mga DEM na nabubuo, na may progresibong lumalaki na bilang ng radius gamit ang inverse distance weighted (IDW) na algorithm at pinagsama. Ang mga natitirang puwang ay pinagsasama gamit ang pinakamalapit na interpolation ng kapitbahay.

Default: 3

--dem-resolution <float>

Resolusyon ng DSM/DTM sa cm / pixel.

Default: 5

--dem-decimation <positive integer>

Tanggalin ang mga puntos bago bumuo ng DEM. Ang "1" ay walang pagkabulok (buong kalidad). Ang "100" naman ay ~ 99% na pagkabulok ng mga puntos. Kapaki-pakinabang para sa pagpapabilis ng henerasyon.

Default: 1

--dem-euclidean-map

Ang map ay nagrereport ng distansiya sa pagitan ng mga cell sa pinakamalapit na NODATA value (bago magkaron ng hole filling). Ito ay magagamit sa pag-isolate ng mga areas na na-fill na.

Default: False

--orthophoto-resolution <float: x ﹥0.0>

Resolusyon ng orthopoto sa cm/pixel.

Default: 5

--orthophoto-no-tiled

I-set ang parameter na ito kung nais ng stripped geoTIFF.

Default: False

--orthophoto-compression <string>

I-set ang compression na gagamitin. Tandaan na maaari nitong ma-break ang gdal_translate kung hindi pamilyar sa iyong ginagawa.

Mga Opsiyon: JPEG | LZW | PACKBITS | DEFLATE | LZMA | NONE

Default: DEFLATE

--orthophoto-bigtiff <string>

I-control kung ang nagawa na orthophoto ay ang BigTIFF o ang classic TIFF. Ang BigTIFF ay para sa files na mas malaki sa 4GB and data. Tignan ang link para sa specs ng GDAL: https://www.gdal.org/frmt_gtiff.html for more info.

Mga Opsiyon: YES | NO | IF_NEEDED | IF_SAFER

Default: IF_SAFER

--orthophoto-cutline

Nag-generate ng poligon sa palligid ng cropping area na nag-cut ng orthophoto sa paligid ng features. ANg poligon na ito ay makakatulong sa pagtatahi ng mossaics na may multiple overlapping orthophotos.

Default: False

--build-overviews

Gumawa ng orthophoto overviews gamit ang gdaladdo.

--verbose, -v

Magprint ng karagdagang mensahe sa console.

Default: False

--time

Mag-generate ng benchmark file sa runtime info.

Default: False

--version

Dinidisplay ang bersiyon ng software number at exits.

--split <positive integer>

Ang average na bilang ng imahe kada submodel. kapag naghihiwalay ng malaking dataset sa mas maliit na submodels, ang mga imahe ay ginugrupo sa clusters. Ang value na ito ay nireregulate ang bilang ng imahe kung saan ang mga cluster ay dapat na may average.

--split-overlap <positive integer>

Radius ng overlap sa pagitan ng submodels. Matapos pagsamasamahin ang mga imahe sa isang cluster, ang mga imahe na mas malapit sa cluster ay nadaragdag agad sa cluster. Ito ay nangyayari para siguraduhin na ang magkakalapit na submodel ay magoverlap.

--optimize-disk-space

Magdelete ng mabibigat na intermediate files (katulad ng original orthos, dtm, dsm) para magamit ng husto ang disk space habang naitatabi ang mga compressed na bersyon. Nakakaapekto ito sa abilidad na magrestart ng ng pipeline mula sa intermediate stage, nagaallow ito ng datasets na magprocess sa machines na walng sapat na space na available. Also, sa mode na ito, ang mga "reports" ay hindi naisusulat sa output 'opensfm' folder.

Default: False

--sm-cluster <string>

URL to a ClusterODM instance for distributing a split-merge workflow on multiple nodes in parallel.

Default: None

--merge <string>

Piliin ang mga imemerge sa merge step ng split dataset. By default, lahat ng output ay minemerge nito.

Default: all
